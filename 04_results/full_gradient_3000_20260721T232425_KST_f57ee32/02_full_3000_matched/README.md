# 공식 Full 3,000-update 기준선

이 디렉터리는 Russian Roulette 실험과 실행 조건을 맞춘 공식 Full 기준선이다.

- `essential/metrics.jsonl`: 3,000개 update의 원시 학습·평가 지표
- `essential/resolved_config.json`: 실제 적용된 설정
- `essential/run_manifest.json`: 실행 환경과 산출물 정보
- `appendix/checkpoints/`: 30, 300, 1,500, 2,700, 3,000 update checkpoint manifest

`90_auxiliary_runs/01_full_3000_original`과 혼동하지 않는다.
